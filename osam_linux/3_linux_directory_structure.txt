리눅스 디렉토리들 설명

/bin :실행파일들(binary), 명령어들이 모여있음
/boot : 부팅과 관련된 것들이 모여있음, 커널관련
/dev : 장치파일(device)들이 모여있음
/etc : 설정파일들이 담겨있음
/home : 일반 사용자들이 모여있음
/lib : 공유 라이브러리들이 들어있음
/lib64 : 64bit 라이브러리들
/media : 미디어 장치들, cd룸 같은 것들
/mnt : 마운트 하는곳
/opt : 리눅스 이외에 사용하는 소프트웨어들
/proc : 프로세스약자, 메모리 상태의 설정값들을 나타냄, 하드디스트가 아니라 메모리 상태임
/root : root의 홈디렉토리
/run : 현재 실행하고 있는 프로세스를 저장
/sbin : superbin, 관리자들이 실행하는 명령어들이 모여있음 
/srv :
/sys : 시스템관련것들이 들어있음
/tmp : 임시디렉토리, 일반사용자들이 home디렉토리 뺴고 사용할 수 없지만, 여기는 사용가능
/usr : unix system resource, 프로그램 설치하면 여기에 들어감     
/var : 가변적인 디렉토리 , 메일, ftp서비스등 다양하게 사용
