원격저장소는 크게 두가지 의미
1.소스코드를 백업한다.
2.다른사람들과 협업한다.


Github를 이용한 원격저장소

1.git의 소스코드를 지역 저장소로 가져오기
fork : 원하는 소스코드를 나한테 가져오는 것 

지역저장소로 가지고 오는 방법
-사이트에서 git clone or download를 눌러 url을 복사하고 
git clone (복사한 url) (새로만들 지역저장소 이름)

2.지역저장소의 소스코드를 github로 올리기
git remote add (이름,대부분 처음은 orgin, master같은 느낌) (주소)
=>의미 : 원격 저장소에 연결하는데(add) 적혀있는 주소로 연결하고, 그 주소는 (이름)으로 부를 것이다. 
했을때 아무것도 안뜨면 성공
git remote하면 가지고 있는 원격저장소가 나온다.
